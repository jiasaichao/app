## Flex组件
### Properties

* `onClick` - click callback
* `label` - button text
* `children` - 
* `className` - 样式名称
* `style` - 样式
* `column` - 垂直排列
* `vertical` - 水平居中对齐
* `HW` - 垂直居中对齐
* `flex1` - flex-grow为1，就是放大倍数为1
* `other`

| propName    | 说明 | propType | defaultValue | isRequired |
|-------------|------|----------|--------------|------------|
| className   | 样式名称 | string     | -            |            |
| style       | 样式 | object   | -            | +          |
| column      | 垂直排列 | bool     | -            |            |
| horizontal  | 水平居中对齐 | bool     | -            |            |
| vertical    | 垂直居中对齐 | bool   | -            | +          |
| HW          | 水平和垂直都居中对齐 | bool     | -            |            |
| flex1       | flex-grow为1，就是放大倍数为1 | bool   | -            | +          |
