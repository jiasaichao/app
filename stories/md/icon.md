## Text 文本组件
### Properties

|    属性    |  说明  |  类型  | 默认值 | 必填 |
| ---------- | ------ | ------ | ------ | ---- |
| color      | 颜色   | string | -      |      |
| width      | 宽度   | string | -      |      |
| height     | 高度   | string |        |      |
| background | 背景色 | string |        |      |
### 源代码
```js
<Page>
      <Icon name='arrowUp' color='#BD10E0' width='.3rem' height='.3rem' background='#F8E71C' />
</Page>
```
